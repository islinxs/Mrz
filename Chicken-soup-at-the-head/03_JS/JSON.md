## JSON
#### Javascript Object Natation 
- 中文翻译: JS对象表示法
- JSON是表示数据的一种格式
- JSON实际就是一个字符串
- JSON中的属性名和属性值必须加双引号
- JSON可以被任意的语言任意的平台所识别
- JSON不能加注释

##### 语法规范
- JSON 对象 '{}'
- JSON 数组 '[]'
- JSON中可以使用的数据类型
	  1. String 字符串
	  2. Numer 数值
	  3. Boolean 布尔值
	  4. null 空值
	  5. Object 对象
	  6. Arr 数组
	  
##### JSON工具类
- JSON.stringify()
	- JS对象转换成JSON字符串
- JSON.parse()
	- JSON字符串解析成JS对象
