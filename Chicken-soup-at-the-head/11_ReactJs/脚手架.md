## React脚手架
##### react脚手架
- 用来帮助程序员快速创建一个基于react库的空项目的库
    - 包含了所有需要的配置
    - 指定好了所有的依赖
    - 可以直接安装/编译/运行一个简单效果
- react提供了一个专门用于创建react项目的脚手架库: create-react-app
- 项目的整体技术架构为: react + webpack + es6 + eslint
  

##### 创建项目并启动
- 全局安装脚手架
    - `npm install -g create-react-app`
- 当前工作目录运行命令
    - `create-react-app 项目名`
- 安装后运行命令
    - `cd 项目名`
- 打包项目
	- `npm build`
- 启动项目
    - `npm start`
